# Profile X

Modernized Profile X web platform with Express backend and SQLite database.

## Features
- User Authentication (Register/Login)
- Article Management
- Dark-themed UI

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   node server.js
   ```

## Technologies
- Node.js
- Express
- SQLite3
- HTML/CSS/JS
